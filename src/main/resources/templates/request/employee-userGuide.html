1. Overview

The Request Workflow lets employees and approvers manage organization-wide requests. You can create drafts, submit requests for approval, attach files, select FAM assets for asset requests, claim expense reimbursements, comment, and monitor progress from the approval timeline.

2. Quick Start

1

Open Requests from the employee sidebar or go to /employee/requests.

2

Click New Request, select a type, and complete the form.

3

Create the draft, review details, then click Submit.

4

Track decisions in the approval timeline, comments, and activity log.

3. Request Types

TypeUse It ForSpecial Rule
Staff RequisitionRequest new or replacement headcount.Requires job and staffing details.
File RequestRequest access to or processing of files/documents.Requires an attachment before submission.
Asset RequestRequest tools, software, equipment, or supplies.Requires a FAM asset selection, quantity, required date, and business justification.
Expense ReimbursementClaim repayment for business expenses you already paid personally.Requires expense lines and a receipt/supporting attachment before submission.
General RequestRequest anything that does not fit the other categories.Requires only common fields.

4. Create And Manage Requests

Create Draft

  1. Click New Request.
  2. Select request type and priority.
  3. Enter title and description.
  4. Fill type-specific fields.
  5. Click Create Draft.

Submit And Track

  1. Open the request detail page.
  2. Add required attachments, select FAM assets for Asset Requests, or add expense lines and receipts for Expense Reimbursements.
  3. Click Submit.
  4. Watch status, comments, and timeline.

5. Approval Flow

Requests are routed through the approval workflow. If a task is assigned to you, it appears under Pending My Approval.

Approve

Moves the request forward or completes approval.

Reject

Stops the workflow. A comment is required.

Return

Sends the request back for correction. A comment is required.

7. Troubleshooting

I cannot create a request

Confirm your login email is linked to an employee record.

No approval route is configured

Ask HR/admin to check your primary reporting line.

File request will not submit

Upload at least one supported attachment first.

Asset request will not submit

Select at least one asset from FAM, enter a valid quantity, required date, and business justification. If the asset list does not load, try again later.

Expense reimbursement will not submit

Add at least one expense line and upload a receipt or supporting document first.
